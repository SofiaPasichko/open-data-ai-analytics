
# Аналіз якості атмосферного повітря (Січень 2026)



**Мета проєкту:**

Проведення аналізу відкритих даних щодо середньодобових концентрацій шкідливих речовин в атмосферному повітрі міст України. 



**Джерело відкритих даних:**

Дані отримано з Державного порталу відкритих даних:

data.gov.ua: https://data.gov.ua/dataset/63cd8827-e6f1-4e0f-8fa1-96f1a5872d9e 



**Гіпотези дослідження:**



*Гіпотеза №1*

Київ демонструє значно вищий рівень концентрації SO2 порівняно з іншими містами через поєднання промислових викидів та масштабного транспортного навантаження мегаполіса.



*Гіпотеза №2:*

Протягом січня рівень забруднення завислими речовинами не демонструє тенденції до зниження, що вказує на стабільне джерело викидів незалежно від погодних умов.



*Гіпотеза №3:*

В промислових центрах показники будуть стабільні і мають малу варіацію через безперервний цикл роботи підприємств.

**Хід роботи**
Було ініціалізацовано Git-репозиторій та створено структуру каталогів. 
Розробка велася із застосуванням такого підходу до контролю версій: кожен етап — від розрахунків до візуалізації — виконувався в окремих гілках, які згодом об'єднувалися з основною гілкою. Це дозволило зберегти детальну історію проекту, яку було зафіксовано першим стабільним релізом версії v0.1.0 з відповідним описом змін у файлі CHANGELOG.

Створено структуру папок (src, data, reports) та налаштовано .gitignore, щоб не засмічувати репозиторій тимчасовими файлами Python та великими датасетами.

У README.md було сформульовано 3 гіпотези (щодо забруднення в Києві, накопичення пилу та аномалій у промислових регіонах).

Через гілку feature/data_load реалізовано механізм отримання даних.

У гілках feature/data_quality_analysis та feature/data_research паралельно велася робота над перевіркою чистоти даних та математичними розрахунками.

Було штучно створено та успішно розв'язано конфлікт у README.md, що імітувало реальну командну роботу, де двоє розробників змінюють один файл.

В останній гілці feature/visualization додано графіки (Lineplot та Boxplot).

Проект зафіксовано у стані v0.1.0. Додано CHANGELOG.md для відстеження історії змін.

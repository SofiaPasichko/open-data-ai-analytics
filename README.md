# Аналіз якості атмосферного повітря (Січень 2026)

**Джерело відкритих даних:**
Дані отримано з Державного порталу відкритих даних (data.gov.ua): [shchodenni-za-sichen-2026.csv](https://data.gov.ua/dataset/63cd8827-e6f1-4e0f-8fa1-96f1a5872d9e)

---

## Опис сервісів та портів

Система складається з таких контейнерів:

| Сервіс | Опис | Порт (Host) |
| :--- | :--- | :--- |
| **db** | База даних PostgreSQL 15 для зберігання показників. | `5432` |
| **data_load** | Скрипт ініціалізації та завантаження даних у БД. | - |
| **data_quality** | Модуль перевірки якості та цілісності даних. | - |
| **data_research** | Обчислення регресії та варіативності забруднення. | - |
| **visualization** | Створення аналітичних візуалізацій (графіків). | - |
| **web** | Flask-застосунок для відображення результатів. | `5000` |

---

##  Команди запуску

Для запуску проєкту необхідно мати встановлений Docker.

1. **Збірка та запуск всіх сервісів:**
   ```bash
   docker compose up --build

2. **Зупинка проєкту:**
   ```bash
   docker compose down

3. **Перегляд результатів:**
http://localhost:5000

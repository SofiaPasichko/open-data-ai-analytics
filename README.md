
# Аналіз якості атмосферного повітря (Січень 2026)



**Мета проєкту:**

Проведення аналізу відкритих даних щодо середньодобових концентрацій шкідливих речовин в атмосферному повітрі міст України. 



**Джерело відкритих даних:**

Дані отримано з Державного порталу відкритих даних:

data.gov.ua: https://data.gov.ua/dataset/63cd8827-e6f1-4e0f-8fa1-96f1a5872d9e 



**Гіпотези дослідження:**



*Гіпотеза №1*

Київ демонструє значно вищий рівень концентрації SO2 порівняно з іншими містами через поєднання промислових викидів та масштабного транспортного навантаження мегаполіса.



*Гіпотеза №2:*

Протягом січня рівень забруднення завислими речовинами не демонструє тенденції до зниження, що вказує на стабільне джерело викидів незалежно від погодних умов.



*Гіпотеза №3:*

В промислових центрах показники будуть стабільні і мають малу варіацію через безперервний цикл роботи підприємств.

# Changelog

## [0.1.0] - 2026-02-26
### Added
- Початкова структура проекту з папкою `src/`.
- Перевірка якості даних (`data_quality.py`).
- Аналіз гіпотез (Київ, Херсон, тренди пилу).
- Модуль візуалізації (`visualization.py`) з графіками Lineplot та Boxplot.
- Датасет щоденних показників за січень 2026.
- Документація в README.md.

## [0.2.0] - 2026-04-15
### Added
- Повна контейнеризація проекту за допомогою **Docker Compose**.
- Мікросервісна архітектура: розподіл функцій між окремими сервісами (db, load, research, web).
- Інтеграція бази даних **PostgreSQL 15** для збереження результатів аналізу.
- Веб-панель керування на **Flask**: реалізовано динамічний перегляд таблиць бази даних та візуалізацій.
- Конфігурація через **.env**: безпечне керування доступами та портами.
- Механізм **Healthchecks**: автоматичне очікування готовності БД перед запуском аналітики.
- Спільні томи (**Shared Volumes**): налаштовано передачу звітів та графіків між контейнерами.
- Оптимізація Docker-образів (використання python-slim) для швидкого розгортання.
- Оновлено README.md: додано інструкції з запуску, опис портів та структуру сервісів.
